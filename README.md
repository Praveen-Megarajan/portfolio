# Praveen Megarajan - Portfolio Website

A clean, responsive portfolio website showcasing my experience as a Software Engineer and SQL Developer.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Minimal Light Theme**: Clean and professional aesthetic
- **Smooth Navigation**: Sticky navigation bar with active section highlighting
- **Interactive Elements**: Copy-to-clipboard functionality for contact information
- **Fast Loading**: Pure HTML/CSS/JavaScript - no frameworks or dependencies

## Sections

- **Hero**: Introduction with profile photo, title, and quick links
- **Core Competencies**: Technical skills organized by category
- **Experience**: Detailed work history with timeline visualization
- **Education**: Academic background and certifications
- **Contact**: Multiple ways to get in touch

## Setup Instructions

### Local Development

1. **Clone or download** this repository
2. **Add your assets**:
   - Place your CV PDF file as `assets/Praveen_Megarajan_Doc.pdf`
   - Place your profile photo as `assets/profile.jpg` (or update the filename in `index.html` if using a different format)
3. **Open** `index.html` in your web browser
   - Simply double-click the file, or
   - Right-click → Open with → Your preferred browser

That's it! No build process or server required.

### Assets Required

- `assets/Praveen_Megarajan_Doc.pdf` - Your CV/resume PDF
- `assets/profile.jpg` - Your profile photo (JPG, PNG, or WebP formats supported)

**Note**: If your profile photo has a different filename or format, update the `src` attribute in the `<img>` tag within the hero section of `index.html`.

## Deployment

### GitHub Pages (Free)

1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to Settings → Pages
4. Select the main branch as the source
5. Your site will be live at `https://yourusername.github.io/repository-name`

### Netlify (Free)

1. Sign up/login at [Netlify](https://www.netlify.com)
2. Drag and drop your project folder onto Netlify
3. Your site will be live instantly with a generated URL
4. Optionally, connect to GitHub for automatic deployments

### Vercel (Free)

1. Sign up/login at [Vercel](https://vercel.com)
2. Import your GitHub repository or upload files
3. Deploy with zero configuration

### Other Hosting Options

- **Any static hosting service**: Upload the files via FTP/SFTP
- **Cloud storage**: AWS S3, Google Cloud Storage, Azure Blob Storage
- **Traditional web hosting**: Upload via cPanel or similar

## Customization

### Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #2563eb;  /* Main accent color */
    --text-primary: #1f2937;   /* Main text color */
    --text-secondary: #6b7280; /* Secondary text color */
    --bg-primary: #ffffff;     /* Background color */
    --bg-secondary: #f9fafb;   /* Secondary background */
}
```

### Content

- Edit `index.html` to update any text content
- Modify sections, add/remove items as needed
- Update contact information in the contact section

### LinkedIn URL

Update the LinkedIn URL in `index.html`:
- Search for `https://www.linkedin.com/in/praveen-megarajan`
- Replace with your actual LinkedIn profile URL

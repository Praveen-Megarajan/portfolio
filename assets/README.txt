ASSETS FOLDER
=============

Please add the following files to this folder:

1. Praveen_Megarajan_Doc.pdf
   - Your CV/resume PDF file
   - This will be downloadable from the portfolio website

2. profile.jpg (or profile.png)
   - Your profile photo/headshot
   - Recommended size: 400x400px or larger (square aspect ratio)
   - Formats supported: JPG, PNG, WebP
   - If using a different filename, update the <img> tag in index.html

Note: The portfolio website will work without these files, but:
- The CV download button won't work until the PDF is added
- A placeholder will show if the profile photo is missing
